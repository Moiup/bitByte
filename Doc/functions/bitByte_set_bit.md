# bitByte_set_bit(\_,\_)

Set a bit to BITBYTE_TRUE.

## **Declaration**

```C
void bitByte_set_bit(
                        byte *b,
                        int i
                    )
```

## **Parameters**
**b**  
A byte.

**i**
The index of the bit to set.

## **Result**
Change the bit `i` int `b` to BITBYTE_TRUE;
