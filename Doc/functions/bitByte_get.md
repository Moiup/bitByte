# bitByte_get(\_,\_)

Returns a bit.

## **Declaration**

```C
void bitByte_get(
                    byte *b,
                    int i
                )
```

## **Parameters**
**b**  
A byte.

**i**
The index of the bit to get.

## **Result**
Returns the bit `i` of the byte `b`.
