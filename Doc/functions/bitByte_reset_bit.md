# bitByte_reset_bit(\_,\_)

Set a bit to BITBYTE_FALSE.

## **Declaration**

```C
void bitByte_reset_bit(
                        byte *b,
                        int i
                    )
```

## **Parameters**
**b**  
A byte.

**i**
The index of the bit to reset.

## **Result**
Change the bit `i` int `b` to `BITBYTE_FALSE`.
