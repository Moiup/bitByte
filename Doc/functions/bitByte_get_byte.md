# bitByte_get_byte(\_,\_)

Returns a byte from a type.

## **Declaration**

```C
void bitByte_get_byte(
                    byte *b,
                    int i
                )
```

## **Parameters**
**b**  
Multiple bytes.

**i**
The index of the byte to get.

## **Result**
Returns the byte `i` from `b`.
